學習日文時，經歷過聽力終於聽懂一點長看的動畫和電視劇的對話後，會漸漸開始想要自己能開始對話了， 這方面的練習，除了找日本朋友或是線上的口語老師之外，也很建議開始開始從寫"廢文"開始，或是廢句子， 腦中想到什麼，都試著寫出來，內容不重要，不夠廢還不行，主題可以是「現在的心情」、「剛剛做的事」、 「今天早上做了什麼？」等等非常簡單的事情，不用真實也可以只是要有個簡單的事情寫即可。

Windows 安裝「日本語」輸入法

以下是 Windows 安裝「日本語」輸入法的安裝步驟

1點選視窗右下角的語言符號。

2然後點選開啟「語言喜好設定」。

如果找不到右下角的語言符號，也可以從與下方式進入「語言喜好設定」:
開始 > 設定 > 時間與語言 > 語言

3點選新增語言，在跳出選擇語言的視窗後

4搜尋或是找到「日本語」，選擇並選下一步後點選安裝

完成!!現在妳可以透過 WIN + 空白鍵切換成日本語輸入法了

沒有特別調整的話預設會是羅馬拼音，試著輸入「ＮＩ　ＨＯ　ＮＮ　ＧＯ」打出日本語吧！
詳細的輸入介紹和快捷鍵在下段會詳細講解

此外，輸入法語言介面區塊下方的「上下鍵」，可以調整語言切換的順序，可以調整成自己習慣的方式。
如果對英文輸入法沒有特殊需求，也可以移除英文輸入法，畢竟中文和日文輸入法都可以輸入英文，這樣可以讓切換語言不用三個語言間循環，但要注意某些軟體在不在英文輸入法快捷鍵會有一點點怪怪的。

Windows 日本語輸入法的操作方法與快捷鍵

Windows 的語言切換快捷鍵

安裝好之後，可以在工具列右下角的輸入法手動切換， 或是使用
「Alt」+「Shift」或「Win」+「空白鍵」切換各語系的輸入法，
看大家習慣選擇一個記住即可，我個人是習慣使用「Win」+「空白鍵」。
＊如果沒看到日語輸入法可以重新安裝看看。

各語言輸入循環切換快捷鍵

日本語輸入法的快捷鍵

開始打字之前我們先熟悉一下， Windows 10 日本語輸入法的切換快捷鍵:
切換至輸入平假名： Ctrl + Caps Lock
切換至輸入片假名：Alt + Caps Lock
切換至輸入日／英：Shift + Caps Lock
雖然可以切換平假名和片假名，但其實到最後發現，輸入跳出的候選選項，大多會跑出正確的假名顯示方式， 比如當你輸入牛奶「MI RU KU 」時，候選選項自然會出現「ミルク」，而不會是「みるく」所以大多時候，不會經常缺換片假名和平假名，但需要時還是很重要。

鍵盤快捷鍵不起作用?好像哪裡怪怪的?

注意如果你的快捷鍵操作方式，跟以上介紹的不太一樣，像是: Caps Lock = 在平假名和英數字元模式之間切換。還有 Alt + Caps Lock 不能切到片假名， 你可能在「日文鍵盤 (106/109 鍵)」模式內，因為我們用的不是日文鍵盤，一般會建議改回「英文鍵盤 (101/102 鍵)」
但不過實際用下來其實也不會有太大差別，甚至因為切換日英文變成只需 Caps Lock ，可能還以較方便，所以就看大家習慣了。

想要接回英文鍵盤的話，可以到「語言喜好設定」:
慣用語言:日本 > 選項 > 硬體鍵盤配置:變更配置 >
選擇「英文鍵盤 (101/102 鍵)」後確定 重新啟動後更改完成

PS. 曾經遇過明明在日文鍵盤卻顯示英文鍵盤的狀況，這時候就只能先變成日文健在變回去了，需要從新開機兩次。

接下來開始打出日文字吧

基本上各家輸入法預設都是使用羅馬拼音，也就是我們學５０音時使用的拼音，大部分按造當初學習時的拼音直接輸入即可。
如果還再熟悉50音嗎的話，推薦 MARUMARU 日語教室，裡面有羅馬拼音和發音，學習上很方面，另外這個網站最知名的是他的聽歌學日文，會將中日文逐句翻譯，更新也非常快，常常正在播的新番或是日劇的歌舊已經更新上去了。

接下來這些舊是比較重要，需要的記的部分，是也很常用的

包括:撥音(ん)、促音（っ）、長音（ー）、小字（ょ）、々〃。

最後是特殊音，特殊音再五十音是沒學到的，但是再學習外來語單字時非常常出現

特殊音是為了讓發音更接近外國語因而產生的，比較新一點時髦的文字都會出現， 如：顯示器 Display「ディスプレー」的「ディ」、Windows＊「ウィンドウズ」的「ウィ」 不過特殊音大多還是都可以猜出來怎麼打，像是 Ｗindow 的 wi 就是ウィ，剛開始練習打字時就算猜不到也可以先打「ｗ」再打上面說的小字打法「li」等到熟悉之後再慢慢記住即可。目前我覺得最難猜到的是 「ディ」的「dhi」畢竟直覺會打「di」結果出來的是「ぢ」。
