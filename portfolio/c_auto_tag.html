Background

Unstructured Data, though challenging to quantify, holds valuable information.

In the process of data analysis, Unstructured Data is often considered one of the most challenging data types to quantify. However, this type of data typically contains rich information. The goal of this project is to utilize Chat GPT to develop a tool for structuring unstructured data in the absence of specific NLP models. We expect this tool to automatically integrate with other data outputs to generate visual charts, thereby enhancing the understanding and presentation of information within unstructured data. This project aims to improve data analysis efficiency while providing a more comprehensive way to present data, ultimately strengthening decision-making and insights.

Process and Method

Data Processing

1. Linking the IDs from the raw data.
2. Removing unnecessary content.

Exploring Chat GPT API Prompts for Data Structuring

1. Generating and testing different Chat GPT query methods to find the most effective approach.
2. Taking the first 100 records and evaluating the TAG reliability and the stability of the returned data structure.
3. Iteratively retesting prompts and establishing debugging procedures for unstable structures.

Extracting Usable Portions Based on the Returned Data Structure

Reconnecting with the original data and creating visualizations.
Testing the automation process and continually sampling to verify reliability.
