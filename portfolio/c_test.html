Usability test for Light Flashing

This interactive webpage serves as a testing tool for exploring the effects of light flickering. The application is designed to help evaluate how individuals perceive different states of a device based on variations in light patterns.
The core feature of the website is a light that can flash at a user-specified interval. It also enables control over the duration of the light being on ("light" in milliseconds) and off ("dark" in milliseconds), as well as the transition time between the two states. A color picker is provided to adjust the light's color, and preset buttons offer common flashing patterns for quick testing.

Interactive Video Switcher Simulation Platform

This webpage provides an interactive interface for video switcher testing, allowing researchers to evaluate the user experience with various front panel button arrangements. It provides controls for a variety of functionalities, including status switch for lights, layout source connector, and user action tracking. The platform supports both manual and logical modes of operation, enabling users to toggle between different views and source combinations using keyboard input. Users can also switch light colors and modes, creating a realistic environment for testing different user interactions. All actions performed on the platform are logged for further analysis, supporting both session-based and individual task-based recording.
